Summary

  1. Data Analysis and Database Creation
    • Analyze the provided data about music composers and create a suitable dataset to be used with JSON-SERVER.
  2. Web Query Service
    • Use CSS and JavaScript to create a webpage that allows querying composers and musical periods.
    • Develop a web service that responds to HTTP GET requests on the following routes:
      • /compositores -> A listing of all composers.
      • /compositores/{id} -> Details about a specific composer.
      • /compositores?periodo={periodo} -> A list of composers from a specific period.
      • /periodos -> A list of all musical periods present in the dataset.
      • /periodos/{id} -> Details about a specific musical period.
  3. CRUD Operations
    • Implement CRUD (Create, Read, Update, Delete) operations for:
      • /compositores/edit -> To add, edit, remove, and query composers.
      • /periodos/edit -> To add, edit, remove, and query musical periods.
